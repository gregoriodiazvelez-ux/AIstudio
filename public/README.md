# Carpeta Pública (public)

Esta es la carpeta donde debes subir tus imágenes estáticas para que sean accesibles en tu sitio web.

## Instrucciones para Imágenes Personalizadas

Para personalizar la sección "Quiénes somos", sube los siguientes archivos a esta carpeta:

1.  **Foto de Familia:**
    *   Nombre del archivo: `foto-familia.jpg`
    *   Se mostrará en la sección "Nuestra Historia" como imagen principal.

2.  **Foto de Alana (Perro):**
    *   Nombre del archivo: `foto-alana.jpg`
    *   Se mostrará en la imagen superpuesta pequeña.

## Instrucciones del Plano

1.  **Plano del Mapa:**
    Sube tu imagen del plano aquí y nómbrala `plano-topografico.jpg` o `plano-topografico.png`.

## Cómo funciona

El código está configurado para buscar estos nombres de archivo específicos. Si no los encuentra, utilizará imágenes de ejemplo automáticamente.
